# Knowledge Base System (Laravel)

A web-based Knowledge Base System built with Laravel and Filament, designed to manage internal documentation in a structured and searchable way.

## Features
- Role-based access control
- Article and category management
- Admin panel built with Filament
- User management
- Searchable knowledge base

## Tech Stack
- Laravel
- Filament
- MySQL

## Purpose
This project is created as a portfolio to demonstrate backend development using Laravel and Filament for internal systems.

## Notes
- This project uses dummy data
- No production or client data included
